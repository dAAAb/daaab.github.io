slant
=====

Extendable 360 Video Player

## install

**component:**

```js
$ component install littlstar/slant
```

## usage

```js
var src = '360-video.mp4';
var el = document.querySelector('#video');
var video = slant.createPlayer(el, {src: src});

video
.render()
.play();
```
